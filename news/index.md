# Changelog

## regtools 0.2.0

- Added a `NEWS.md` file to track changes to the package.
- Added
  [`plot_map()`](https://amslala.github.io/regtools/reference/plot_map.md)
  function to create Norwegian choropleth municipality and country maps
- Added extra check in
  [`synthetic_data()`](https://amslala.github.io/regtools/reference/synthetic_data.md),
  due to some incompatibility issues with KlassR it currently requires
  stringr v1.5.1
