# Package index

## Main functions

- [`read_diag_data()`](https://amslala.github.io/regtools/reference/read_diag_data.md)
  : Read and validate the structure of diagnostic data
- [`read_admin_data()`](https://amslala.github.io/regtools/reference/read_admin_data.md)
  : Read and validate the structure of administrative (sociodemographic)
  individual-level data
- [`filter_diag_data()`](https://amslala.github.io/regtools/reference/filter_diag_data.md)
  : Validate and filter diagnostic data by selected ICD-10 or ICPC-2
  codes
- [`filter_admin_data()`](https://amslala.github.io/regtools/reference/filter_admin_data.md)
  : Filter administrative (sociodemographic) data by selected filtering
  parameters
- [`curate_diag_data()`](https://amslala.github.io/regtools/reference/curate_diag_data.md)
  : Curate diagnostic data
- [`link_diag_admin()`](https://amslala.github.io/regtools/reference/link_diag_admin.md)
  : Link diagnostic and demographic datasets using unique personal
  identifiers
- [`calculate_incidence()`](https://amslala.github.io/regtools/reference/calculate_incidence.md)
  : Calculate incidence rates
- [`calculate_prevalence()`](https://amslala.github.io/regtools/reference/calculate_prevalence.md)
  : Calculate prevalence rates
- [`calculate_prevalence_series()`](https://amslala.github.io/regtools/reference/calculate_prevalence_series.md)
  : Calculate prevalence series rates
- [`calculate_incidence_series()`](https://amslala.github.io/regtools/reference/calculate_incidence_series.md)
  : Calculate incidence series rates
- [`plot_rates()`](https://amslala.github.io/regtools/reference/plot_rates.md)
  : Plot prevalence/incidence rates
- [`plot_map()`](https://amslala.github.io/regtools/reference/plot_map.md)
  : Plot spatial data

## Helper functions

- [`synthetic_data()`](https://amslala.github.io/regtools/reference/synthetic_data.md)
  : Create synthetic diagnostic, time-varying and time-invariant
  individual level data
- [`get_population_ssb()`](https://amslala.github.io/regtools/reference/get_population_ssb.md)
  : Get yearly population data from SSB, by region, sex and age.
- [`harmonize_municipality_codes()`](https://amslala.github.io/regtools/reference/harmonize_municipality_codes.md)
  : Harmonize old municipality codes in Norway to 2024 municipality
  codes
