# Norwegian municipalities valid in 2016

Data frame including Norwegian municipality codes valid in 2016.

## Usage

``` r
kommuner_2016
```

## Format

`kommuner_2016` A data frame with 429 rows and 2 columns:

- code:

  Code of municipality (2016)

- name:

  Name of municipality
